# XML/YML to XLSX Converter

Небольшое desktop-приложение на Python для конвертации XML/YML-прайсов в Excel (`.xlsx`). Скрипт умеет читать файл с диска или загружать прайс по URL, извлекать товары и категории, а затем сохранять результат в Excel с отдельными листами `Товары` и `Категории`.

Приложение использует `tkinter` для GUI, `pandas` для табличной обработки и `openpyxl` как Excel-движок при записи `.xlsx`.

## Возможности

- Загрузка XML/YML из локального файла.
- Загрузка XML/YML по URL.
- Автоопределение кодировки XML или принудительный выбор кодировки.
- Исправление HTML/ESC-сущностей вроде `&amp;`, `&nbsp;`, `&eacute;` перед разбором XML.
- Два режима обработки повторяющихся полей: объединение в один столбец или разворачивание в отдельные колонки.
- Экспорт в Excel с автоподбором ширины колонок.
- Отдельный лист для категорий.

## Для чего подходит

Скрипт полезен для интернет-магазинов, поставщиков и интеграций, где прайс приходит в XML/YML, а анализировать его удобнее в Excel. `xml.etree.ElementTree` входит в стандартную библиотеку Python и подходит для разбора XML, а `openpyxl` поддерживает чтение и запись файлов формата `.xlsx`.

## Требования

- Python 3.10+
- `tkinter` — стандартный интерфейс Python к Tcl/Tk и обычно доступен на Windows и macOS, а на Linux может потребовать отдельной установки через пакетный менеджер.
- `pandas` — работа с таблицами.
- `openpyxl` — запись Excel `.xlsx`.
- `pyperclip` — вставка URL из буфера обмена.

## Установка

```bash
pip install pandas openpyxl pyperclip
```

```bash
python yml2xlsx.py
```

## Как пользоваться

1. Выберите источник данных: локальный файл или URL.
2. Укажите XML/YML-файл или ссылку на прайс.
3. Выберите папку сохранения.
4. Укажите имя выходного `.xlsx`-файла.
5. При необходимости задайте кодировку XML.
6. Выберите режим обработки повторяющихся полей:
   - **merge** — значения объединяются через `;`
   - **separate** — создаются отдельные колонки (`picture_1`, `picture_2` и т.д.)
7. Оставьте включённой опцию исправления ESC/HTML-сущностей, если прайс может содержать `&nbsp;`, `&eacute;`, `&copy;` и подобные значения.
8. Нажмите **Конвертировать**.

## Что на выходе

На выходе создаётся Excel-файл с двумя листами:

- `Товары` — основная таблица с офферами.
- `Категории` — список категорий из XML/YML.
